About Campus Life Planner

Campus Life Planner helps students organize tasks, track time commitments, and manage their academic workload effectively.

Features

  • Add, edit, and delete tasks with validation
  • Track task duration and due dates
  • Organize with custom tags
  • Advanced regex search functionality
  • Sort by date, title, or duration
  • Visual statistics dashboard
  • Weekly time cap monitoring
  • Import/Export data as JSON
  • Fully keyboard accessible
  • Mobile-responsive design

Keyboard Shortcuts

  • Tab - Navigate forward
  • Shift + Tab - Navigate backward
  • Enter / Space - Activate buttons
  • Escape - Cancel edit mode
